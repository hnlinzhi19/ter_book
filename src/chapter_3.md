# 五大过程

## 整体管理

| 标题     | 整体管理           |
| -------- | ------------------ |
| 启动过程 | 制定项目章程       |
| 规划过程 | 制定项目管理计划   |
| 执行过程 | 指导与管理项目工作 |
| 监控过程 | 监控项目工作       |
|          | 实施整体变更控制   |
| 收尾过程 | 结束项目或阶段     |
